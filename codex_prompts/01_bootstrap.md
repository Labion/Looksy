# Codex Prompt 01: Bootstrap

请先阅读仓库根目录的 `AGENTS.md`、`README.md` 和 `docs/` 下的所有文档。

目标：为「看展搭子」创建第一个可运行的个人 Web App 原型。

请完成以下任务：

1. 在 `apps/web` 中创建 Next.js + TypeScript 应用。
2. 配置 Tailwind CSS。
3. 创建移动端优先的首页。
4. 首页需要包含：
   - App 名称：看展搭子
   - 一句话说明：个人 AI 看展笔记助手
   - 主操作：新建看展记录
   - 次操作：查看历史记录
5. 添加基本目录结构：
   - `app/`
   - `components/`
   - `lib/`
   - `types/`
6. 暂时不要实现数据库、AI 调用或图片上传。
7. 更新根目录或 app 内 README，说明如何安装依赖、运行开发服务器。

验收标准：

- 本地可以启动开发服务器。
- 首页可以在手机宽度下正常显示。
- 没有硬编码 API key。
- 没有引入不必要的复杂依赖。

完成后请说明：

1. 改了哪些文件。
2. 如何运行。
3. 如何测试。
4. 下一步建议。
